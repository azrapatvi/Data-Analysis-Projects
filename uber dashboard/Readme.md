# Uber Analytics Dashboard

This repository contains an interactive Power BI dashboard built using Uber ride data. The dashboard provides insights into bookings, revenue, customers, vehicle performance, and overall ride behavior.

## Overview

The dashboard includes analysis of:
- Total bookings
- Completed bookings
- Cancelled/Lost bookings
- Revenue analysis
- Customer behavior insights
- Vehicle-type performance
- Driver and customer ratings
- Top pickup and drop locations
- Monthly and quarterly revenue trends

The design includes:
- A mobile-style sliding filter panel (created using bookmarks)
- A 3D-style vehicle selection bar
- Dynamic Month/Quarter toggle using Field Parameters
- Clean, minimal UI inspired by Uberâ€™s color theme
- Synced slicers across all pages

## Key Features

### Mobile Sliding Filter Drawer
A left-side filter panel that opens and closes like a mobile app navigation drawer.

### Dynamic Parameter Toggle
Switch between Monthly and Quarterly revenue insights using Power BI Field Parameters.

### 3D Vehicle Selector
Clickable visuals for Auto, Bike, Go Mini, Premier Sedan, and Uber XL.

### Modern UI
Minimalistic layout with bold headers, clean visuals, and high readability.

### Synced Slicers
All slicers work seamlessly across the dashboard pages.

## Insights Provided

### Vehicle Metrics
- Completed vs Lost bookings
- Revenue by vehicle type
- Average ride distance
- Pickup and drop hotspots
- Trend of monthly bookings

### Customer Insights
- Revenue by customer
- Payment method distribution
- Customer rating patterns

### Revenue Analysis
- Total revenue
- Monthly and quarterly trends
- Revenue contribution per vehicle type

## Tools Used

| Tool | Purpose |
|------|---------|
| Power BI Desktop | Dashboard development |
| Power Query | Data cleaning and shaping |
| DAX | Calculations and measures |
| Field Parameters | Month/Quarter toggle |
| Bookmarks | Sliding filter navigation |


## How to Use

1. Download the `Uber_Dashboard.pbix` file.
2. Open it using Power BI Desktop.
3. Navigate through pages and use slicers, toggles, and filters to explore insights.
